# Dev

## Try it out

Starlog uses both [npm](https://www.npmjs.com/) and [Bower](http://bower.io/) to install dependencies. If you do not have them, you'll need to install them first.

```
$ bower install
$ npm install
$ npm run serve
# open in localhost:8082
```
